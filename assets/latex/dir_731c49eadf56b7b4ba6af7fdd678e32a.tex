Information on features and functionality are documented and can be found in the {\ttfamily \doxylink{ak8975_8h}{ak8975.\+h}} header file and in the {\ttfamily documentation} folder.\hypertarget{README.md_autotoc_md10}{}\doxysubsection{\texorpdfstring{Repository}{Repository}}\label{README.md_autotoc_md10}
