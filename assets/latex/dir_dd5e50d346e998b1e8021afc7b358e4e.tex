Information on features and functionality are documented and can be found in the {\ttfamily \doxylink{bh1750_8h}{bh1750.\+h}} header file and in the {\ttfamily documentation} folder.\hypertarget{README.md_autotoc_md22}{}\doxysubsection{\texorpdfstring{Repository}{Repository}}\label{README.md_autotoc_md22}
The component is hosted on github and is located here\+: \href{https://github.com/K0I05/ESP32-S3_ESP-IDF_COMPONENTS/tree/main/components/peripherals/i2c/esp_bh1750}{\texttt{ https\+://github.\+com/\+K0\+I05/\+ESP32-\/\+S3\+\_\+\+ESP-\/\+IDF\+\_\+\+COMPONENTS/tree/main/components/peripherals/i2c/esp\+\_\+bh1750}}\hypertarget{README.md_autotoc_md23}{}\doxysubsection{\texorpdfstring{General Usage}{General Usage}}\label{README.md_autotoc_md23}
