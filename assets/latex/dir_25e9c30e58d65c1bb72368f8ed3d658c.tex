This ESP32 espressif IoT development framework (esp-\/idf) i2c peripheral driver was developed for the OSRAM AS7341 11-\/channel multi-\/spectral digital sensor. Information on features and functionality are documented and can be found in the {\ttfamily \doxylink{as7341_8h}{as7341.\+h}} header file and in the {\ttfamily documentation} folder.\hypertarget{README.md_autotoc_md18}{}\doxysubsection{\texorpdfstring{Repository}{Repository}}\label{README.md_autotoc_md18}
The component is hosted on github and is located here\+: \href{https://github.com/K0I05/ESP32-S3_ESP-IDF_COMPONENTS/tree/main/components/peripherals/i2c/esp_as7341}{\texttt{ https\+://github.\+com/\+K0\+I05/\+ESP32-\/\+S3\+\_\+\+ESP-\/\+IDF\+\_\+\+COMPONENTS/tree/main/components/peripherals/i2c/esp\+\_\+as7341}}\hypertarget{README.md_autotoc_md19}{}\doxysubsection{\texorpdfstring{General Usage}{General Usage}}\label{README.md_autotoc_md19}
