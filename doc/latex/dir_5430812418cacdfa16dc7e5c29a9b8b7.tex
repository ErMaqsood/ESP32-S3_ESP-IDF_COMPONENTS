This ESP32 espressif IoT development framework (esp-\/idf) i2c peripheral driver was developed for the Inven\+Sense MPU6050 6-\/axis motion tracking sensor. Information on features and functionality are documented and can be found in the {\ttfamily \doxylink{mpu6050_8h}{mpu6050.\+h}} header file and in the {\ttfamily documentation} folder.\hypertarget{README.md_autotoc_md70}{}\doxysubsection{\texorpdfstring{Repository}{Repository}}\label{README.md_autotoc_md70}
The component is hosted on github and is located here\+: \href{https://github.com/K0I05/ESP32-S3_ESP-IDF_COMPONENTS/tree/main/components/peripherals/i2c/esp_mpu6050}{\texttt{ https\+://github.\+com/\+K0\+I05/\+ESP32-\/\+S3\+\_\+\+ESP-\/\+IDF\+\_\+\+COMPONENTS/tree/main/components/peripherals/i2c/esp\+\_\+mpu6050}}\hypertarget{README.md_autotoc_md71}{}\doxysubsection{\texorpdfstring{General Usage}{General Usage}}\label{README.md_autotoc_md71}
