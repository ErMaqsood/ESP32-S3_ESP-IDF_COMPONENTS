This ESP32 espressif IoT development framework (esp-\/idf) i2c peripheral driver was developed for the Sensirion SHT4X series of sensors (SHT40, SHT41, SHT43, and SHT45). Information on features and functionality are documented and can be found in the {\ttfamily \doxylink{sht4x_8h}{sht4x.\+h}} header file and in the {\ttfamily documentation} folder.\hypertarget{README.md_autotoc_md78}{}\doxysubsection{\texorpdfstring{Repository}{Repository}}\label{README.md_autotoc_md78}
The component is hosted on github and is located here\+: \href{https://github.com/K0I05/ESP32-S3_ESP-IDF_COMPONENTS/tree/main/components/peripherals/i2c/esp_sht4x}{\texttt{ https\+://github.\+com/\+K0\+I05/\+ESP32-\/\+S3\+\_\+\+ESP-\/\+IDF\+\_\+\+COMPONENTS/tree/main/components/peripherals/i2c/esp\+\_\+sht4x}}\hypertarget{README.md_autotoc_md79}{}\doxysubsection{\texorpdfstring{General Usage}{General Usage}}\label{README.md_autotoc_md79}
To get started, simply copy the component to your project\textquotesingle{}s {\ttfamily components} folder and reference the {\ttfamily \doxylink{sht4x_8h}{sht4x.\+h}} header file as an include. The component includes documentation for the peripheral such as the datasheet, application notes, and/or user manual where applicable.
