\doxysubsection{Detailed Description}
\href{https://github.com/K0I05}{\texttt{ }} \href{/LICENSE}{\texttt{ }} \href{https://en.wikipedia.org/wiki/C_(programming_language)}{\texttt{ }} \href{https://docs.espressif.com/projects/esp-idf/en/stable/esp32/index.html}{\texttt{ }} \href{https://code.visualstudio.com/}{\texttt{ }} \href{https://platformio.org/}{\texttt{ }} \href{https://registry.platformio.org/libraries/k0i05/esp_ssd1306}{\texttt{ }} \href{https://components.espressif.com/components/k0i05/esp_ssd1306}{\texttt{ }}

This esp-\/idf driver was developed for generic SSD1306 OLED displays. Information on features and functionality are documented and can be found in the {\ttfamily \doxylink{ssd1306_8h}{ssd1306.\+h}} header file. The SSD1306 component is a compact and simplified driver compatible with 128x64, 128x32, 128x128 OLED displays. There are three font sizes supported, hardware and software scrolling capabilities, bitmap visualization, and more. This component has one font implemented now (i.\+e. 8x8 basic Latin + control + extended Latin) but is ideal for most use cases.\hypertarget{README.md_autotoc_md82}{}\doxysubsection{\texorpdfstring{Repository}{Repository}}\label{README.md_autotoc_md82}
The component is hosted on github and is located here\+: \href{https://github.com/K0I05/ESP32-S3_ESP-IDF_COMPONENTS/tree/main/components/peripherals/i2c/esp_ssd1306}{\texttt{ https\+://github.\+com/\+K0\+I05/\+ESP32-\/\+S3\+\_\+\+ESP-\/\+IDF\+\_\+\+COMPONENTS/tree/main/components/peripherals/i2c/esp\+\_\+ssd1306}}\hypertarget{README.md_autotoc_md83}{}\doxysubsection{\texorpdfstring{General Usage}{General Usage}}\label{README.md_autotoc_md83}

\hypertarget{README.md_autotoc_md84}{}\doxysubsection{\texorpdfstring{Basic Example}{Basic Example}}\label{README.md_autotoc_md84}
Once a driver instance is instantiated the display panel is ready for usage as shown in the below example. This basic implementation of the driver utilizes default configuration settings and displays a sequence of text messages and bitmaps at user defined interval and prints the results.

The example initializes a 128x64 SSD1306 OLED display and demonstrates the following features\+:


\begin{DoxyItemize}
\item Display large text (x3)
\item Display file receive and transmit bitmap icons
\item Display medium text (x2)
\item Display text
\item Display countdown timer
\item Display text scrolling up
\item Display text scrolling down
\item Display text paging down and up
\item Display text scrolling horizontally from right and left
\item Display text scrolling vertically downwards and upwards
\item Display bitmap images
\item Display inverted text and fadeout
\end{DoxyItemize}
