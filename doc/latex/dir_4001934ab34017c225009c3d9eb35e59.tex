Information on features and functionality are documented and can be found in the {\ttfamily \doxylink{mmc56x3_8h}{mmc56x3.\+h}} header file and in the {\ttfamily documentation} folder.\hypertarget{README.md_autotoc_md66}{}\doxysubsection{\texorpdfstring{Repository}{Repository}}\label{README.md_autotoc_md66}
The component is hosted on github and is located here\+: \href{https://github.com/K0I05/ESP32-S3_ESP-IDF_COMPONENTS/tree/main/components/peripherals/i2c/esp_mmc56x3}{\texttt{ https\+://github.\+com/\+K0\+I05/\+ESP32-\/\+S3\+\_\+\+ESP-\/\+IDF\+\_\+\+COMPONENTS/tree/main/components/peripherals/i2c/esp\+\_\+mmc56x3}}\hypertarget{README.md_autotoc_md67}{}\doxysubsection{\texorpdfstring{General Usage}{General Usage}}\label{README.md_autotoc_md67}
