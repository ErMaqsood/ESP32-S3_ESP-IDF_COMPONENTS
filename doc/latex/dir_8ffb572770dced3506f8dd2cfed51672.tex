Information on features and functionality are documented and can be found in the {\ttfamily \doxylink{mlx90614_8h}{mlx90614.\+h}} header file and in the {\ttfamily documentation} folder.\hypertarget{README.md_autotoc_md62}{}\doxysubsection{\texorpdfstring{Repository}{Repository}}\label{README.md_autotoc_md62}
