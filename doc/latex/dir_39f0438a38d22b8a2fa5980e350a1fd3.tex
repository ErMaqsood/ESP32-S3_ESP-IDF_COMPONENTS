This ESP32 espressif IoT development framework (esp-\/idf) i2c peripheral driver was developed for the Honeywell HMC5883L 3-\/axis digital compass sensor. Information on features and functionality are documented and can be found in the {\ttfamily \doxylink{hmc5883l_8h}{hmc5883l.\+h}} header file and in the {\ttfamily documentation} folder.\hypertarget{README.md_autotoc_md50}{}\doxysubsection{\texorpdfstring{Repository}{Repository}}\label{README.md_autotoc_md50}
The component is hosted on github and is located here\+: \href{https://github.com/K0I05/ESP32-S3_ESP-IDF_COMPONENTS/tree/main/components/peripherals/i2c/esp_hmc5883l}{\texttt{ https\+://github.\+com/\+K0\+I05/\+ESP32-\/\+S3\+\_\+\+ESP-\/\+IDF\+\_\+\+COMPONENTS/tree/main/components/peripherals/i2c/esp\+\_\+hmc5883l}}\hypertarget{README.md_autotoc_md51}{}\doxysubsection{\texorpdfstring{General Usage}{General Usage}}\label{README.md_autotoc_md51}

The component includes documentation for the peripheral such as the datasheet, application notes, and/or user manual where applicable.
